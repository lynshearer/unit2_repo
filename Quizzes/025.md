## Working Solution to Quiz #025

```.py
from matplotlib import pyplot as plt

x_out = []
y_out = []
st = -10

for num in range(101):
    x_out.append(st)
    y_out.append(abs(st))
    st += 0.2

plt.xlabel('x')
plt.ylabel('f(x) = |x|')
plt.plot(x_out, y_out, color="red")
plt.show()
```
